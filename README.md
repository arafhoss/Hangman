# Hangman Game (C++)

## My Game
- A simple command-line Hangman game built in C++.  
- The player must guess a randomly selected word one letter or word at a time before running out of attempts.

---

## Features
- Random word selection
- ASCII Hangman visual stages
- Limited number of guesses (lives system)
- Input validation
- Win / lose detection
- Simple console-based gameplay

---

## How to Run

### 1. Compile the program
```bash
g++ main.cpp -o hangman
```
### 2. Run the program
#### Windows:
hangman.exe
#### Mac / Linux:
./hangman
